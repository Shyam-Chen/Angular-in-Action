# Git

```bash
$ git add .

$ git add <FILE_NAME>
```

```bash
$ git status
```

```bash
$ git log

$ git log -1
```

```bash
# remove tracked file
$ git rm --cached <FILE_NAME>

# remove tracked folder
$ git rm --cached -r <FOLDER_NAME>
```

```bash
$ git stash

$ git stash list

$ git stash apply
```

```bash
$ git cherry-pick <COMMIT_ISH>
```
